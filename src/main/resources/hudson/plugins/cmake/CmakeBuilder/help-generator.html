The build tool chosen by CMake — corresponding to the generator — is exposed in the CMAKE_MAKE_PROGRAM build environment variable. Subsequent build steps may use $CMAKE_MAKE_PROGRAM to execute the generated scripts.
Its actual value is retrieved from the generated CMake cache file (CMakeCache.txt). Usually that value is something like /usr/bin/gmake on Linux.
